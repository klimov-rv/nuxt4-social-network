# Argito_worldwide

Argito — глобальная P2P-платформа для частных объявлений. Аналог Avito с международным охватом и поддержкой криптовалют

## 🚀 Быстрый старт

### Требования

- Node.js 20.18+
- npm 9+

### Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Или для продакшена
npm run build
npm run preview
```

**Доступ по адресу:** `http://localhost:3000`

---

## 🛠 Технологический стек

- **Фреймворк:** Nuxt 3
- **Язык:** TypeScript
- **UI библиотека:** Element Plus
- **Тестирование:** Vitest
- **Стили:** SCSS + CSS Variables

## 🏗 Архитектура проекта

```

├── app/                # Основной код приложения

│   │   ├── widgets/    # FSD | Крупные составные блоки

│   │   ├── entities/   # FSD | Бизнес-сущности
│   │   └── shared/     # FSD | Переиспользуемый код
│   ├── providers/
│   │   ...
│   └── app.vue
└── pages/              # Страницы (Nuxt routing + FSD layer)
   ...

```
